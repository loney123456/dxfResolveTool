# dxfResolveTool
dxf文件解析工具

编译运行环境：VS2010IDE、Window10


本项目是本人在实习期间，在公司制作的一个小项目，基于C++语言和MFC框架的dxf文件解析小工具。


主要的功能有：增删图形，删除部分图形，缩放、移动图形还有对齐、镜像图形和将图形进行排序等功能。

具体功能演示的视频链接：https://www.iqiyi.com/v_19tfhh9j55c.html  密码：1176311321
